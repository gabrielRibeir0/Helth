# Helth

## Contribuir

- Verificação do código

`ruff check`

- Corrigir erros

`ruff check --fix`

- Fazer pull request

```sh
git checkout -b nome-branch

git add .
git commit -m "<msg>"

git push -u origin nome-branch
```

- Apagar branch local (e remoto no Github)

```sh
git checkout main
git branch -d nome-branch
```
