# ADRs (Architecture Decision Records)

Esta pasta contém os registos formais de decisões arquiteturais do projeto Helth.

## Como usar

- Cada ADR deve ser numerado sequencialmente: `0001`, `0002`, `0003`, ...
- Nome sugerido: `000X-titulo-curto.md`
- Estado sugerido: `Proposto`, `Aceite`, `Substituído`
